A full stack application that replicates the core functionalities of a modern brokerage platform, focusing on real-time data streaming, portfolio analytics, and real-time stock monitoring.


Tech Stack

Frontend: React.js, Vite

Styling: Tailwind CSS (for a clean, minimalist "Kite" inspired UI)

State Management: Redux Toolkit (handling complex global states like user funds and holdings)

Backend: Node.js, Express.js

Database: MongoDB

Real-Time: Socket.io / WebSockets

Authentication: JWT (JSON Web Tokens)

Key Features

Real-Time Price Updates: Integration with WebSockets (Socket.io) to provide live ticker updates without page refreshes.

Dynamic Dashboard: Overview of market trends, top gainers, and losers.

Order Management: Buy and Sell functionality with logic for market and limit orders.

Interactive Charts: Visualization of historical and real-time stock data.
